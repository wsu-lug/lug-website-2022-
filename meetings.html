Free as in speech, not as in beer. The Linux Users' Group at WSU

Meetings

Our meetings are approximately every week. Our meetings generally consist of students, alumni, industry professionals, or professors discussing a technical topic that is of interest to the club. At some of these meetings, we also have officer elections, discussions about where the club is going, and much more. If you have technical topic suggestions, please contact us via email or on social media.

Upcoming Meetings - Tuesdays @ 6:00 in Spark 223

September 6th - How To Use Linux

Not sure what a terminal is? And what distro is the best? We'll try and show you how to use a few Linus distro's and you'll see how easy and not so daunting it is! FREE PIZZA!!

September 13th - The Linux Programming World

How do I code in Linux? Where does C++ and Python fit into this mess? We'll walk through some good IDE's to use and ones not to use, as well as give some examples on how coding some configuration files can change your experience. FREE PIZZA!!

September 20th - Git Some Programs

GitHub and git are powerful tools for open source developers. We'll skim the surface on how to use git on Linux and what you can use it for. FREE PIZZA!!

September 27th - What Makes A Linux?

Systemd, Gnome, KDE, Mate, Linux-PAM, Kernel's, GRUB? What the heck are all these things? Never fear, LUG is here! We'll show you some system utilities and how they run and make your system work! FREE PIZZA!!
